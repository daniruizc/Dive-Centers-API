# DiveCenters API

> Backend API for DiveCenters application, which is a dive centers directory website

## Usage
Duplicate file "config.env" to ".env" and update the values/settings to your own

## Install Dependencies
```
npm install
```

## Run application
```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
